## Variable Test (value)
```python
var1 = 123
#$VARIABLETEST simple-test
#$TESTVAR var1
#$PROPERTY value 123
```

## Variable Test (evalString)
```python
var1 = 123
#$VARIABLETEST simple-test
#$TESTVAR var1
#$PROPERTY evalString "120 + 3"
```
